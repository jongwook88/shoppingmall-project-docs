[금주 프로젝트 진행]

1. 쇼핑몰 프론트 구현
> 메인 > 회원가입 > 로그인 > 상품상세 > 장바구니 담기 > 장바구니 목록(옵션)

2. 쇼핑몰 관리자 구현
> 쇼핑몰 관리자 로그인 > 상품 목록 > 상품 등록
> 쇼핑몰 관리자 로그인 > 회원 목록
> 쇼핑몰 관리자 로그인 > 장바구니 목록(옵션)


제출 방법
1. 위 구현 구간을 동영상으로 제작하여 제출
2. 동영상 촬영시 access log 를 tail 하여 동영상 안에 포함

제출 일시
1차: 8월 8일 자정까지
2차: 8월 12일 09시 이전까지
